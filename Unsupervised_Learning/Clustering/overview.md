# Clustering Overview

Clustering is an unsupervised learning technique that groups similar data points together without predefined labels.

## Key Concepts

- **Similarity Metric**: Distance measures (Euclidean, Manhattan, etc.)
- **Cluster**: Groups of similar data points
- **Centroid**: Center of a cluster

## Algorithms Covered

1. **KMeans**: Partitions data into K clusters using centroids
2. **Hierarchical Clustering**: Creates a tree of clusters (dendrogram)
3. **DBSCAN**: Density-based clustering that finds clusters of arbitrary shapes
4. **Gaussian Mixture Models**: Probabilistic model assuming data comes from mixture of Gaussians

## Use Cases

- Customer segmentation
- Document categorization
- Image compression
- Pattern discovery

## When to Use Each

- **KMeans**: Fast, scalable, works well with spherical clusters
- **Hierarchical**: Good for dendrogram visualization
- **DBSCAN**: Better for non-spherical clusters and outlier detection
- **GMM**: When probabilistic approach is needed
